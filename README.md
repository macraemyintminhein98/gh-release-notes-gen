# APEX: GH Release Notes Generator

Automatically generate semantic release notes for your GitHub projects. 

This GitHub Action streamlines your release process by parsing conventional commits and producing a well-formatted changelog. 

**Features:**
- Conventional Commit parsing
- Customizable output
- Easy to integrate

**Usage:**
Add `apex-products/gh-release-notes-gen@v1` to your GitHub Actions workflow.
